1 Generating the permutation distribution

1.1 Using infer

Beerwings %>%
  specify(Hotwings ~ Gender) %>%
  hypothesize(null = "independence") %>%
  generate(reps = 10000, type = "permute") %>%
  calculate(stat = "diff in means", order = c("M", "F")) -> permDist
permDist %>%
  visualize() +
  theme_bw() + 
  geom_vline(xintercept = c(-obs_diff, obs_diff))

pvalue <- mean(permDist$stat >= obs_diff)*2
pvalue

1.2 Using for()

B <- 10000
diff_means <- numeric(B)
for(i in 1:B){
  diff_means[i] <- diff(tapply(Beerwings$Hotwings, sample(Beerwings$Gender), mean))
}
hist(diff_means)
abline(v = c(-obs_diff, obs_diff))

pvalue <- mean(diff_means >= obs_diff)*2
pvalue
[1] 0.0022

Based on a p-value of 0.0022 which is less than \(\alpha = 0.05\) we reject the null hypothesis and conclude that there is evidence to suggest the average wings consumed by males is not the same as the average number of wings consumed by females.

2 Constructing 95% Bootstrap Percentile CI

2.1 Using infer

Beerwings %>% 
  specify(Hotwings ~ Gender) %>% 
  generate(reps = 10000, type = "bootstrap") %>% 
  calculate(stat = "diff in means", order = c("M", "F")) -> bootdist
bootdist %>% 
get_confidence_interval(level = 0.95, type = "percentile") -> PCI

We are 95% confident that the true difference between the mean hotwings consumed by males and the mean hotwings consumed by females fall in the interval \((2.3 \text{ hotwings}, 7.9 \text{ hotwings})\).
